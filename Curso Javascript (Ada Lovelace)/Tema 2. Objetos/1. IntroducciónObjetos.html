Introducción a Objetos en Javascript

Datos primitivos

Tipos

  • Definición: puede ser una cadena de caracteres, números, booleanos, null o undefined.

    • let cadena = "Gabriel González";
    let pi = 3.14;
    let bool = true; //false
    let noDefinida = undefined;
    let nula = null;

Objetos

  • Definición: no dependen del navegador.
      String
      Number
      Boolean
      Date
      Math
      RegExp (Expresiones regulares)
      Array
      Function
      Object

Objetos de alto nivel

  • Definición: algunos dependen del navegador como son:
      Window
      Screen
      Navigator
      Location
      History
      Document

  • Un objeto contiene:
      Propiedades: se definen como nombre/valor
      Métodos: acciones que se pueden ejecutar sobre un objeto
      Pueden contener uno o mas valores (datos primitivos, otros objetos o funciones).
