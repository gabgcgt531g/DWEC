Sentencias if...else en Javascript

If

  • Definición de If: es una estructura de control, que realizan acciones en función de las acciones que tomamos. Es una sentencia condicional.
    • -> Especifica un bloque de código que se ejectua si una condición es cierta.
  • Sintaxis de If:

    • If (condición) {
    'Instrucciones se si la condición es verdadera'
    }
    • -> error

    • No cumple con la condición

Else

  • Definición de Else: es una estructura de control, siempre acompañada de If, que se ejecuta si la condición del If es falsa.
  • Sintaxis de Else:

    • If (condición) {
    'Instrucciones se si la condición es verdadera'
    } else {
    'Instrucciones se si la condición es falsa'
    }
    • -> error


Else if

  • Definición de Else If: especifica una nueva condición si la primera es falsa.
  • Sintaxis de Else If:

    • If (condición) {
    'Instrucciones se si la condición es verdadera'
    } else if {
    'Instrucciones se si la condición 1 es falsa y la condición 2 es veradera'
    } else {
    'Instrucciones se si todas las condiciones que la preceden son falsas'
    }
    • -> error

    • -> error
