Sentencias break y continue en Javascript

break y continue

break

  • Definición: hace que salgas de un bucle, finalizandolo.

  • Sintáxis:se puede inlcuir en bucles, como ya hemos visto, como son switch, o for.

    • var i;
    for (i = 0; i < 10; i++){
    if(i == 3){
    break;
    }
    document.write(i);
    }

    El resultado es:
    Sale del bucle cuano se cumple la condición que ejecuta el break.

continue

  • Definición: hace saltar hasta la sigueinte iteración del bucle. Permite saltar una o mas iteraciones.

  • Sintáxis:se puede inlcuir en bucles, como ya hemos visto, como son switch, o for.

    • var j;
    for (j = 0; j < 10; j++){
    if(j == 3){
    continue;
    }
    document.write(j);
    }

    El resultado es:
    Con la sentencia continue, salta la iteración, que le marca la condición, hasta la siguiente instrucción.

Etiquetas

  • break en etiquetas: preceden a una sentencia con el nombre de la etiqueta, seguido de dos puntos. Si utilizamos varias sentencias en una etiqueta, las englobaremos entre llaves.

    • var animales = ["perro, ", "gato, ", "hamster"];
    ver_animales: {
    document.write(animales[0]);
    document.write(animales[1]);
    break ver_animales;
    document.write(animales[2]);
    }

    Un break sin etiqueta, debe estar dentro de un bucle o un switch. Si no ponemos etiqueta, falla el código.
