Operadores de tipo en Javascript

Javascript tiene dos modelos de operadores de tipo, que nos van a permitir comprobar de que tipo es la variable que hemos declcarado.
Hay dos tipos:
  • typeoff. Nos devuelve el tipo de dato de una variable o dato.
  • instanceof. Nos devuelve true si un objeto es una instacia de otro especificado.
