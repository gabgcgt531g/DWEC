Sentencia do...while en Javascript

do...while

  • Definición: difiere del bucle while, que al menos se ejecuta una vez. Se ejecutará hasta que la condición sea falsa.

  • Sintáxis:

    • do {
    //Código a ejecutar
    } while(condicion);

    -> condición: definimos la condición para que, cuando se cumpla, salga del bucle.

    var i = 0;
    do {
    document.write(i);
    } while(i < 10);
