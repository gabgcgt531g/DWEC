Ámbito de variables en Javascript

Variables locales.

  • Características:
    • -> Se definen dentro de una función.
    -> Tienen ámbito local en la función.
    -> Son accesibles únicamnte desde dentro de la función.
    -> Podemos declarar variables con el mismo nombre en diferentes funciones.
    -> La variable local desparece cuando acaba la función.
    • -> error

    • -> error

Variables globales.

  • Características:
    • -> Se definen fuera de las funciones.
    -> Tienen ámbito global, en toda la página.
    -> Son accesibles únicamnte desde dentro y fuera de la función.
    -> La variable global desparece cuando acaba la página.
    • -> error

Variables automáticamente globales.

  • Características:
    • -> Si asginamos un valor a una variable no declarada, se convierte en global.
    • -> error

Variables globales y locales con el mismo nombre.

  • Características:
    • -> Podemos utilizar el mismo nombre para una variable local y una global, pero la manera de llamarlas difiere. Veremos a continucaión.
    • -> error

Variable global redefinada dentro de una función.

  • Características:
    • -> Si asginamos una variable automáticamente global, dentro de la función, la variable de dentro de la función pisa a la de fuera, por orden de compilación.
    • -> error
