Sentencia For en Javascript

For

  • Definición de For: Nos permite ejecutar el código un número determinado de veces.

  • Sintaxis de For:

    • for (sentencia_1; sentencia_2; sentencia_3) {
    //Código que quermemos que se repita
    }

    -> sentencia_1: se ejecuta antes de que empiece el bloque de código a repetir.
    -> sentencia_2: define la condición por la que el bloque de código se va a ejecutar.
    -> sentencia_3: se ejecuta después de que acabe el bloque de código a repetir.

    var i;
    for (i = 0; i < 5; i++) {
    document.write(i);
    }

Variaciones.

  1. Si omitimos la primera sentencia, deberemos haber inicializado las variable con anterioridad.

    2. var j;
    for (; j < 5; j++) {

  2. Si omitimos la segunda sentencia, dentro del código, a repetir, debemos incluir una instrucción break.

    3. var k;
    for (k = 1; ; k++) {
    document.write(k);
    if (k == 5) {
    break;
    }
    }

  3. Si omitimos la tercera sentencia, debemos incrementar o decrementar el valor de la variable dentro del bucle.

    4. var l;
    for ( l= 1; l <= 5; ) {
    document.write(l);

  4. Se puede trabajar con dos o mas valores dentro del bucle.

    5. var m, n;
