Sentencia While en Javascript

While

  • Definición: a diferencia del bucle for, permite ejecutar código mientras se cumpla una condición, no le indicamos las veces exactas .

  • Sintáxis:

    • while (condición) {
    //Código a ejecutar
    }

    -> condición: definimos la condición para que entre el bucle.

    var i = 0;
    while (i < 10) {
    document.write(i);
    i++;
    }
