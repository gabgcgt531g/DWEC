Sentencia For...in en Javascript

For..in

  • Definición: Nos permite recorrer arrays y listas.

  • Sintáxis:

    • for (sentencia_almacena_valores in sentencia_valor_array) {
    //Código para que muestre valores
    }

    -> sentencia_almacena_valores: en ella almacenarmos cada valor que muestre, del array.
    -> sentencia_array:indica el nombre del array del queremos traer sus valores.

    var animal = {nombre: "Paco", tipo: "perro", raza: "Snauzer", edad: 12};
    var propiedad;
    for (propiedad in animal) {
