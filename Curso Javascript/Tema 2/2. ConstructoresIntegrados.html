Constructores integrados en Javascript

Definición de Objetos.

Objetos con homólogo primitivo: nos permite construir cadenas. Al tener un dato primitivo String no es recomendable utilizarlo como objeto.
Esto se puede aplicar también para Number o Boolean por la misma razón.

Tipo Array

  • let x = new Array();
  • let x = [];

    • Tipo RegExp

  • let x = new RegExp();
  • let x = /()/;

    • Tipo Function

  • let x = new Function();
  • function();

    • Tipo Date

  • let x = new Date();

    • Tipo Math

    No se puede declarar con new el objeto Math porque es un objeto global.
    Para llamarlo, ponermos math.nombreExpresion.
