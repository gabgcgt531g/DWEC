Sentencias en Javascript

Tipos

  • Punto y coma (;).
    • Es recomendable que las sentencias acaben en punto y coma (;).
  • Escritura en la misma linea.
    • Se pueden poner varias sentencais en la misma línea, separnadolas por punto y coma (;).
  • Espacios en blanco.
    • Los espaciones en blanco se ignoran. Se aconsejan en operaciones matemáticas
  • Líneas de mas de 80 caracteres.
    • Es recomendable dividir la linea después de un operador.
  • Bloques de código.
    • Se agrupan entre llaves.
    Se utilizan en funciones, bucles, condicinoes, etc.

Palabras reservadas (keywords).

  • break
  • continue
  • switch
  • for
  • do...while
  • if...else
  • debugger
  • try...catch
  • var
  • function
  • return
