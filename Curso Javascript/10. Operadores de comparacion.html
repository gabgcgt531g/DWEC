Operadores de comparación en Javascript

Se utilizan para determinar la igualdad o desingualdad entre valores o variables.
El resultado dependera del tipo de la variable.

Tipos:

-> Operador de igualdad (==):
    Comprara el valor.

-> Operador de igualdad estricta (===):
    Comprara el valor y el tipo de dato.

-> Operador de desigualdad (!=):
    Comprara el valor.

-> Operador de desigualdad estricta (!==):
    Comprara el valor y el tipo de dato.

-> Operador mayor que (>):
    Comprara dos valores.

-> Operador menor que (<):
    Comprara dos valores.

-> Operador mayor o igual que (>=):
    Comprara dos valores.
    
-> Operador menor o igual que (<=):
    Comprara dos valores.
