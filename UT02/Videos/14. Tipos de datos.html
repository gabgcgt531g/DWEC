Tipos de datos en Javascript

Javascript no es un lenguaje fuertemente tipado, por lo que, al sumar, por ejemplo, una cadena y un número, no mostrará error al compilarlo, simplemente, entenderá a ambos como una cadena y los pondrá en orden de izquierda a derecha.
Vemaos los tipos existentes:
  1. String (cadena). Se definen entre comillas simples ('') o dobles ("").
    • =>>
  2. Number (números). Pueden ser enteros o decimales (se separa con un punto(.)).
  3. Notación cientifica (exponencial). Un ejemplo sería 'var x = 12e5;', donde 12 es la base y 5 es el número de ceros que le siguen. Si el número después de 'e' es negativo, se dividirá entre el número de ceros que indique dicho número.
    • =>>
  4. Booleanos. Sus valores son true o false y se utilizan para evaluar condiciones.
    • =>>
  5. Array (arreglos). Conjunto de valores que se almacenan en una estructura. Los valores se guardan entre corchetes ([]). Se pueden almacenar diferentes tipos de datos.
    • =>>
  6. Object (objetos). Ya hay objetos en Javascript predefinidos. Pero nosotros podemos crearlos, en vez de con corchetes con llaves, introduciendo pares clave-valor.
    • =>>
  7. typeof. Nos muestra el tipo de dato de cada variable u función.
    • =>>
  8. Undefined. Nos sirve para controlar los tipos de datos que no están definidos. Cuando una varialbe tiene Undefined pierde el tipo.
    • =>>
  9. Empty (vacio). Indica que una variables tiene un tipo de dato definido pero ningún valor. Se utilza sobre en cadenas.
    • =>>
  10. Null. No pierde la identidad del tipo de dato, pero indica que su valor es nulo.
