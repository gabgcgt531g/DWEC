Tipos de datos en Javascript

Javascript no es un lenguaje fuertemente tipado, por lo que, al sumar, por ejemplo, una cadena y un número, no mostrará error al compilarlo, simplemente, entenderá a ambos como una cadena y los pondrá en orden de izquierda a derecha.
Vemaos los tipos existentes:
  • String (cadena). Se definen entre comillas simples ('') o dobles ("").
    • =>>
  • Number (números). Pueden ser enteros o decimales (se separa con un punto(.)).
  • Notación cientifica (exponencial). Un ejemplo sería 'var x = 12e5;', donde 12 es la base y 5 es el número de ceros que le siguen. Si el número después de 'e' es negativo, se dividirá entre el número de ceros que indique dicho número.
    • =>>
  • Booleanos. Sus valores son true o false y se utilizan para evaluar condiciones.
    • =>>
  • Array (arreglos). Conjunto de valores que se almacenan en una estructura. Los valores se guardan entre corchetes ('var lista = [];'). Se pueden almacenar diferentes tipos de datos.
    • =>>
  • Object (objetos). Ya hay objetos en Javascript predefinidos. Pero nosotros podemos crearlos.
    • =>>
  • typeof. Nos muestra el tipo de dato de cada variable u función.
    • =>>
  • Undefined. Nos sirve para controlar los tipos de datos que no están definidos.
    • =>>
